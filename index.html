Game Instructions:

  • The game starts with 4 black buttons.
  • Turns are alternating between computer and player.
  • First round the computer will randomly select one button and change the color from black to yellow.
  • It is now the players turn to click that same button.
  • If the player selects the correct button, the game will advance to the next round.
  • Each round will increment by an additional button changing to yellow.
    Example: Round 2, computer turn will select 2 buttons, one after the other and turn yellow. The player will then have to select those 2 same buttons and so forth.
  • The game is won if the player can keep up with 10 random selections!

    • GOOD LUCK!
